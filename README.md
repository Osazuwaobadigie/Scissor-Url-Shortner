## About the Project

Scissor App is designed to help users shorten URLs, redirect links, generate QR codes, and analyze link usage with ease and efficiency.


## Features

- URL Shortening
- Custom Alias Creation
- Redirection Tracking
- QR Code Generation
- Analytics Dashboard
